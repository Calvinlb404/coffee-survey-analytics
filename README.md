### How to run
- Create & activate python virtual env (optional; for mac)
  - `python3 -m venv .venv`
  - `source ./venv/bin/activate`
- Install libraries
  - `pip install -r ./requirements.txt`
- Analyze ☕️ 
  - Edit `Input` section 
  - Run all cells
